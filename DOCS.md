# Task Scheduler Pro

Schedule automated tasks for your Home Assistant system with a beautiful, easy-to-use interface.

## Quick Start

1. Click **Open Web UI** to access the scheduler
2. Click **+ New Task** to create your first scheduled task
3. Choose an action, set a schedule, and save!
